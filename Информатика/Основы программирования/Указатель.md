## Определение
**Указатель** (от англ. *pointer*) — это [[переменная]], которая хранит адрес в [памяти](Виртуальная%20память.md) другой переменной. Указатели позволяют работать с данными напрямую через их адреса в памяти, что даёт возможность эффективно управлять памятью и создавать сложные структуры данных, такие как связанные списки, деревья и графы.

---
## Синтаксис
```cpp
// Объявление указателя
int* ptr;           // указатель на int
double* dptr;       // указатель на double

// Получение адреса переменной (оператор &)
int x = 42;
int* ptr = &x;      // ptr теперь хранит адрес переменной x

// Разыменование указателя (оператор *)
int value = *ptr;   // получаем значение переменной, на которую указывает ptr
*ptr = 100;         // изменяем значение переменной через указатель
```

---
## Пример использования
```cpp
#include <iostream>

int main() {
    int a = 10;
    int* ptr = &a;  // ptr указывает на переменную a
    
    std::cout << "Значение a: " << a << std::endl;           // 10
    std::cout << "Адрес a: " << &a << std::endl;             // адрес в памяти
    std::cout << "Значение ptr: " << ptr << std::endl;       // адрес a
    std::cout << "Значение *ptr: " << *ptr << std::endl;     // 10
    
    *ptr = 20;  // изменяем значение a через указатель
    
    std::cout << "Новое значение a: " << a << std::endl;     // 20
    
    return 0;
}
```

## Динамическое выделение памяти
```cpp
// Выделение памяти для одного элемента
int* p = new int(42);
delete p;  // освобождение памяти

// Выделение памяти для массива
int* arr = new int[10];
delete[] arr;  // освобождение памяти массива

// Пример с указателем на структуру
struct Node {
    int data;
    Node* next;
};

Node* node = new Node{10, nullptr};
delete node;
```

---
## Нулевой указатель
```cpp
int* ptr = nullptr;  // указатель не указывает ни на что

if (ptr == nullptr) {
    std::cout << "Указатель пустой" << std::endl;
}
```

---
## Основные операции
- `&` — взятие адреса переменной
- `*` — разыменование (получение значения по адресу)
- `new` — динамическое выделение памяти
- `delete` — освобождение динамически выделенной памяти
- `nullptr` — нулевой указатель (C++11)

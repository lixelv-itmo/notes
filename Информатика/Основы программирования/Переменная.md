## Определение
**Переменная** - это именованная область [памяти](Виртуальная%20память.md) для хранения данных определённого типа. Переменная имеет имя, тип и значение.

---
## Синтаксис
```cpp
тип имя = значение;

int x = 10;           // целое число
double pi = 3.14;     // число с плавающей точкой
char c = 'A';         // символ
bool flag = true;     // логическое значение
std::string s = "hi"; // строка
```

---
## Пример
```cpp
#include <iostream>

int main() {
    int age = 25;           // объявление и инициализация
    age = 26;               // изменение значения
    
    std::cout << age << std::endl;  // вывод: 26
    
    return 0;
}
```

## Модификаторы и области видимости

### const
Константная переменная - её значение нельзя изменить после инициализации.
```cpp
const int MAX_SIZE = 100;
// MAX_SIZE = 200;  // ошибка компиляции

const double PI = 3.14159;
```

### static
Статическая переменная сохраняет своё значение между вызовами функции.
```cpp
void counter() {
    static int count = 0;  // инициализируется только один раз
    count++;
    std::cout << count << std::endl;
}

counter();  // 1
counter();  // 2
counter();  // 3
```

### extern
Позволяет использовать глобальную переменную, объявленную в другом файле.
```cpp
// file1.cpp
int globalVar = 42;

// file2.cpp
extern int globalVar;  // объявляем, что переменная определена где-то ещё
std::cout << globalVar << std::endl;  // 42
```

### Глобальные переменные
Переменные, объявленные вне функций, доступны во всей программе.
```cpp
int globalCounter = 0;  // глобальная переменная

void increment() {
    globalCounter++;    // доступна в любой функции
}

int main() {
    increment();
    std::cout << globalCounter << std::endl;  // 1
    return 0;
}
```

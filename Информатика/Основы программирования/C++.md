## Определение
**C++** - это компилируемый, статически типизированный язык программирования общего назначения, созданный Бьёрном Страуструпом в 1985 году как расширение языка C. Поддерживает процедурное, объектно-ориентированное и обобщённое программирование.

---
## Ключевые особенности

### Производительность
- Прямой доступ к памяти
- Минимальные накладные расходы
- Один из самых быстрых языков

### Парадигмы программирования
- **Процедурное** - функции и структуры
- **ООП** - классы, наследование, полиморфизм
- **Обобщённое** - шаблоны (templates)
- **Функциональное** - лямбда-функции, алгоритмы STL

### Возможности
- Ручное управление памятью
- Множественное наследование
- Перегрузка операторов
- RAII (Resource Acquisition Is Initialization)
- Стандартная библиотека шаблонов (STL)

---
## Области применения

- **Системное программирование** - операционные системы, драйверы
- **Игровые движки** - Unreal Engine, Unity (частично)
- **Высокопроизводительные приложения** - трейдинг-системы, научные вычисления
- **Встраиваемые системы** - микроконтроллеры, IoT
- **Графика и 3D** - рендеринг, компьютерное зрение
- **Базы данных** - MySQL, MongoDB, PostgreSQL

## Стандарты C++

| Стандарт | Год | Основные нововведения |
|----------|-----|----------------------|
| C++98 | 1998 | Первый стандарт ISO |
| C++03 | 2003 | Исправления ошибок |
| C++11 | 2011 | auto, lambda, умные указатели, nullptr |
| C++14 | 2014 | Улучшения C++11 |
| C++17 | 2017 | std::optional, structured bindings |
| C++20 | 2020 | Модули, концепты, ranges, coroutines |
| C++23 | 2023 | std::print, expected, mdspan |

---
## Преимущества и недостатки

### ✅ Преимущества
- Высокая производительность
- Полный контроль над ресурсами
- Богатая стандартная библиотека
- Обратная совместимость с C
- Широкое применение в индустрии

### ❌ Недостатки
- Сложность изучения
- Ручное управление памятью (если не использовать умные указатели)
- Медленная компиляция
- Отсутствие встроенной сборки мусора
- Легко допустить ошибки

## Основные концепции

### Типы данных
```cpp
int, double, char, bool, string
```

### Управляющие конструкции
```cpp
if/else, switch, for, while, do-while
```

### ООП
```cpp
class MyClass {
public:
    void method();
private:
    int data;
};
```

### Шаблоны
```cpp
template<typename T>
T max(T a, T b) {
    return (a > b) ? a : b;
}
```

### STL (Standard Template Library)
```cpp
std::vector, std::map, std::string, std::algorithm
```

## Компиляция и запуск
```bash
# GCC/G++
g++ -std=c++17 main.cpp -o program
./program

# Clang
clang++ -std=c++17 main.cpp -o program

# MSVC (Visual Studio)
cl /std:c++17 main.cpp
```

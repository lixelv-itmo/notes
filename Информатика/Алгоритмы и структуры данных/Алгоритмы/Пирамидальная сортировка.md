## Определение
Пирамидальная сортировка ([англ.](https://ru.wikipedia.org/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA "Английский язык") Heap sort) — это алгоритм неустойчивой сортировки, основанный на структуре данных [[Двоичная куча|двоичная куча]]. Алгоритм состоит из двух этапов: сначала строится max-heap (максимальная куча) из исходного массива, затем элементы поочередно извлекаются из вершины кучи и помещаются в конец отсортированной части массива, после чего восстанавливается свойство кучи.

---
## Сложность и визуализация

- **Худшее время:** $O(n \log n)$
- **Лучшее время:**   $O(n \log n)$
- **Среднее время:**  $O(n \log n)$
- **Затраты памяти:** $O(1)$

## Реализация на C
```c
void swap(int* a, int* b) {
	int temp = *a;
	*a = *b;
	*b = temp;
}

void heapify(int arr[], int n, int i) {
	int largest = i;
	int left = 2 * i + 1;
	int right = 2 * i + 2;
	
	if (left < n && arr[left] > arr[largest])
		largest = left;
	
	if (right < n && arr[right] > arr[largest])
		largest = right;
	
	if (largest != i) {
		swap(&arr[i], &arr[largest]);
		heapify(arr, n, largest);
	}
}

void heap_sort(int arr[], int n) {
	for (int i = n / 2 - 1; i >= 0; i--) {
		heapify(arr, n, i);
	}
	for (int i = n - 1; i > 0; i--) {
		swap(&arr[0], &arr[i]);
		heapify(arr, i, 0);
	}
}
```

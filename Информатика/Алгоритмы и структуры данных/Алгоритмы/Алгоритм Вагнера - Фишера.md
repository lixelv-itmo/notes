## Определение
Алгоритм Вагнера - Фишера - это один из алгоритмов [динамического программирования](Динамическое%20программирование.md) нахождения расстояния Левенштейна между 2 разными строками.
## Определение i, j элемента dp
```c
dp[i][j] = 0 ? i == 0 && j == 0 :
		   i * delete_cost * ? i > 0 && j == 0 :
		   j * insert_cost ? i = 0 && j > 0 :
		   dp[i - 1][j - 1] ? s1[i] = s2[j] :
		   
		   min(dp[i - 1][j] + delete_cost,
		   dp[i][j - 1] + insert_cost,
		   dp[j - 1][i - 1] + replace_cost)
```
